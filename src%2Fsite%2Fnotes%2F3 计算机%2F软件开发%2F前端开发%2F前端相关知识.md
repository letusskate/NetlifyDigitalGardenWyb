## Node.js
在服务器端构建一个可以执行 javascript 代码的环境。  
也就是前端程序员也可以利用服务器执行相关的关于后端的任务

## Vue.js
浏览器端的 javascript 框架。  
利用 vue.js 可以使用封装化的 javascript 构建浏览器页面，更简洁。
