linux 与其他电脑传输数据比较麻烦，无法一个软件（如微信）实现多端传输

## 与 android 传输
可以下载 youqilin 微信，传输文件  
两台设备都登陆 qq，传输文件即可。

## 与 windows 传输
可以下载 youqilin 微信，传输文件  
两台设备都登陆百度网盘，通过同步空间传输即可。因为不能两台设备都登陆 qq。  
两台设备都登陆坚果云也可以，可以公用一个文件夹  
两台设备还可以通过 obsidian 传输文字
