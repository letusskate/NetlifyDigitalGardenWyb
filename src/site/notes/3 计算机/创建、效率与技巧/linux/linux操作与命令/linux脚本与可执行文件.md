### bash 和 sh 的区别
[Shell bash和sh区别 - 一只小菜菜鸟 - 博客园](https://www.cnblogs.com/conefirst/articles/15232323.html)  
sh 遵循 POSIX 规范：“当某行代码出错时，不继续往下解释”。bash 就算出错，也会继续向下执行。
