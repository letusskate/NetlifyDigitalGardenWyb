## 添加环境变量
### 方法一
#### 步骤
~/.bashrc 文件末尾加上一行
```shell
export PATH=$PATH:/home/LS/Cpp/Paracooba-master/build/
```
冒号后面是需要添加到 path 的文件夹路径。
#### 例子
export PATH=$PATH:/usr/local/texlive/2023/bin/x86_64-linux
