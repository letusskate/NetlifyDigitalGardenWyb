## 取消文件锁
从系统文件夹复制到用户目录时，会使得文件有锁头。  
是因为该文件属于 root 用户。  
因此可以改变文件的所有者来消除锁头。
```
sudo chown 用户名 文件名
```
对于目录
```
sudo chown -R 用户名 目录名
```
