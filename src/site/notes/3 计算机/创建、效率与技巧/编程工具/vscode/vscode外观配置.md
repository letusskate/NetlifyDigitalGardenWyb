## 大屏显示器设置
在 `.vscode` 目录下的 `settings.json` 进行配置

```
      //以下为大屏显示器设置  

    "editor.fontSize": 18,  

    "window.zoomLevel": 1,
```

## 简体中文
安装 extension  
crtl+shift+p 输入 display，设置语言即可。
