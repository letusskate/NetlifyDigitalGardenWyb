## github 是什么
github 提供的是一个代码管理平台。  
里面有很多代码，可以支持通过 git 进行提交和克隆等操作。

## github project
project 可以和某些仓库关联  
可以管理 issue 或者是自己添加的待做、在做和已完成  
可以和 pull request 关联  
可以用作项目进度管理

## github issue
使用软件的人或者开发者可以提出对软件该进建议或者 bug。  
也可以自己给自己写未来的规划，但是自己使用的话，不如直接使用 project。

### Milestone
