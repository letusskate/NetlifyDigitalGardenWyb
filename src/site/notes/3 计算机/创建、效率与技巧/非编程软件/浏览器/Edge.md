## 快捷键
Crtl+w 关闭当前标签页  
Crtl+n 新建标签页  
百度界面，/直接进入搜索框。  
crtl+k，用默认搜索引擎搜索

## edge 设置
搜索栏可以设置默认搜索引擎  
new tab 可以设置为仅有一个搜索框  
安全性 - 关闭 Microsoft defender smartscreen（可以更快开始下载）  
edge://flags- 开启 parallel downloading（可以类似 idm 快速多线程下载）
