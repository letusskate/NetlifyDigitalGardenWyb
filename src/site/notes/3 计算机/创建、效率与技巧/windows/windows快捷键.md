shift+ 鼠标滚轮，左右平移  
win+q 搜索  
win+ 数字 任务栏快捷执行  
win+l 锁屏  
win+d 返回桌面  
win+e 资源管理器  
win+r 执行  
win+crtl+d 创建新桌面  
win+crtl+ 左右 切换桌面  
win+crtl+f4 关闭当前桌面  
Win+shift+s 截图  
Win+w 小组件  
W+g xbox game bar 可以录屏  
Crtl+shift+esc 任务管理器  
crtl+alt+del 三大金刚键 cad  
alt+shift+numloc，小键盘当鼠标。5 代表左键，\* 代表右键  
新建文件夹：crtl+shift+n
